# Gen4Regen

**Addressing Training Data Scarcity using Image Generators:
The Gen4Regen Dataset for Forest Regeneration Mapping**

This research framework addresses the bottleneck of expert-annotated data in forestry by leveraging generative AI to boost semantic segmentation performance. By integrating synthetic imagery and pixel-aligned masks generated by the Gemini Nano Banana Pro model, the methodology significantly improves the identification of rare tree species in complex regeneration zones.
